# Workspace

This repository includes a minimal Snake game written with Pygame.

## Running the Game

1. Install `pygame` (requires Python 3):
   ```bash
   pip install pygame
   ```
2. Run the game:
   ```bash
   python main.py
   ```

Use the arrow keys to control the snake. Avoid hitting the walls or yourself while
trying to eat the red food squares. The game window size and speed can be
changed by editing the values in `snake_game/game.py`'s `Config` dataclass.
